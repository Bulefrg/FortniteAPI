Fortnite Status API

This API provides information about the status of Fortnite, Epic Games Store, Rocket League, and more.

Endpoints:

  • /api/fortnite_status - Get Fortnite status

    Retrieve the current status of Fortnite services.

  • /api/epic_games_store_status - Get Epic Games Store status

    Retrieve the current status of the Epic Games Store.

  • /api/rocket_league_status - Get Rocket League status

    Retrieve the current status of Rocket League.

  • /api/epic_online_services_status - Get Epic Online Services status

    Retrieve the current status of Epic Online Services.

  • /api/unreal_engine_status - Get Unreal Engine status

    Retrieve the current status of Unreal Engine.

  • /api/epic_website_status - Get Epic Website status

    Retrieve the current status of the Epic Games website.

  • /api/developer_portal_status - Get Developer Portal status

    Retrieve the current status of the Epic Games Developer Portal.

  • /api/twinmotion_status - Get Twinmotion status

    Retrieve the current status of Twinmotion.

  • /api/rocket_league_sideswipe_status - Get Rocket League Sideswipe status

    Retrieve the current status of Rocket League Sideswipe.

  • /api/lego_fortnite_status - Get LEGO Fortnite status

    Retrieve the current status of LEGO Fortnite.

  • /api/support_a_creator_status - Get Support-A-Creator status

    Retrieve the current status of the Support-A-Creator program.

  • /api/ue_marketplace_status - Get UE Marketplace status

    Retrieve the current status of the Unreal Engine Marketplace.

  • /api/scrape_images - Scrape Images

    Scrape image URLs from a specified page.

Usage:

Make a GET request to the desired endpoint to retrieve the status information.
